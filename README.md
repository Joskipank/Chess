# Chess

Учебный проект на С++: реализация шахмат с консольным интерфейсом с расширением в будущем веб интерфейсом и модульной архитектурой

## Цели
- Реализовать модульную архитектуру с четким разделением на независимые модули: доска, фигуры, ход, игра, рендер 
- проектирование API
- Получить рабочую MVP

## Структура проекта
- `src/`
  - `main.cpp` — точка входа и запуск игрового цикла.
  - `game.cpp` — состояние партии, очередь ходов, условия завершения.
  - `board.cpp` — модель доски и операции над полями.
  - `piece.cpp` — базовая модель фигуры.
  - `move.cpp` — представление хода и его свойства.
  - `rules.cpp` — проверка легальности ходов и условий (шах/мат и т.п.).
  - `renderer.cpp` — вывод состояния доски (консоль/GUI).
- `include/` — заголовки для классов и интерфейсов.
- `tests/` — модульные тесты.

## План работ (этапы)
1. Модель доски и фигур.
2. Базовая генерация и проверка ходов (без сложных правил).
3. Шах/мат, рокировка, взятие на проходе, превращение пешки.
4. Тесты и стабилизация.
5. Улучшение UI (при необходимости).

## Запуск
Сборка будет оформлена через CMake (будет добавлено позже).
